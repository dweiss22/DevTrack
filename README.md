# DevTrack

DevTrack is an application for tracking data related to a team's development of online course content. It is intended to help teams monitor course production work, capture key development details, and maintain visibility into progress across course-building workflows.

## Purpose

Online course development often involves many moving parts: course titles, owners, statuses, deadlines, review cycles, production notes, and handoffs between team members. DevTrack provides a central place to track that information so teams can make better decisions and keep course work moving.

## Core Use Cases

- Track online course content development across a team.
- Monitor course status, ownership, and progress.
- Record development notes, blockers, and production details.
- Support planning, review, and reporting for course creation workflows.
